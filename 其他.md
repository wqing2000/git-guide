# 其他

* 安装

  ```bash
  #详情见官网下载页
  https://git-scm.com/downloads
  ```

* `git archive --format=zip <branch-name> > <file_name>` 打包

  ```bash
  git  archive --format=zip master > nb.zip
  ```

* `--` 转义符

  ```bash
  git checkout -- master
  #前面加上‘--’代表master是一个文件，不是master分支
  ```

* `git help <command>` 查看帮助

  ```bash
  git help -a
  git help add
  ```

  https://fhefh2015.github.io/Fast-GitHub/
